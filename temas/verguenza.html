Vergüenza


Para que Dios confiese su ausencia.
Para que la calle no huela a tristeza.
Para que el miedo no pise la escuela.
Para que el silencio derrote a las sirenas.

Para que la emoción cure la indiferencia.
Para que los invisibles no enloquezcan de pena.

TANTO CREES, TANTO PUEDES
TANTO EMPUJAS, TANTO VENCES

TANTA VERDAD Y TANTA VERGÜENZA

Para que las leyes no cambien la historia.
Para que la poesía desarme a la violencia.
Para que la fe mueva montañas.
Para que la indignación ocupe la mañana.

Para que las preguntas persigan respuestas.
Para que la justicia se quite la venda.

Para que no se cierre la boca de la calle.
Para que la pobreza no sea una raza.
Para que los conversos descubran sus dudas.
Para que la tormenta encuentre las tumbas.

Para que el futuro no vaya a su entierro.
Para que los desheredados pidan su herencia.
